Stardew Valley Mods

2021, Completed

Telephone Purchasing changes the way the player can interact with the in-game phone item by allowing them to purchase items through it. Items are normally only purchasable through the in-game stores scattered through the world, with the telephone item allowing the player to browse the items being sold at these stores.

CommunityCenterAnywhere changes the way the player can interact with the in-game "Community Center" menu by allowing them to insert items into the bundles. Bundles inside of this menu normally do not allow the player to place an item into their respective slots.

Stingbot

2022, Completed

Stingbot is an all-purpose discord bot that tracks various statistics of guild members that are not otherwise tracked. Using a JSON file to store the data, Stingbot is able to use the discord.py API in order to communicate to the Discord servers and relay the information back to a local or private server. Stingbot is solely in development for the remainder of 2022.

Super Mario Bros. (NES) 1-1

2021, Completed

Super Mario Bros. (NES) 1-1 is a 1:1 recreation of level 1-1 from Super Mario Bros. (NES) in Unreal Engine 4. It uses Unreal Engine 4's 2D game creation template and online assets to look as faithful to the original game as possible.

AppleCatch

2021, Completed

AppleCatch is a game coded in C++ using the SFML library where the goal is to catch as many apples as possible without running out of lives. The game uses all unique assets (except for the background music).
